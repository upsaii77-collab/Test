README

■Technical specifications

-Works on Unity Versions : 2021.3.31f1 or higher.
-Texture size : 512 × 512 to 2048 × 2048 (Mostly 1024 × 1024)
-Total Polygons : △4.3M (If you deactivate the books △1.7M)
-Types of materials : PBR
-Types of texture maps : BaseMap, Normal, MaskMap
-LOD information : No
-RenderPipeline : Compatible with Unity Standard Pipeline , You can easely converted to URP(Universal Render Pipeline).


■The package contains

- 1 DemoScene
- 25 FBX
- 26 UniquePrefabs
- 63 Textures
- 33 Materials
- 1 Shader(MatCapShader)
